# 📡 Telecom Customer Churn Analysis — Excel Dashboard

An end-to-end Excel analytics project analyzing customer churn behavior for a telecom company — identifying *who* is leaving, *why* they leave, and *what patterns* predict churn.

---

## 📊 Project Overview

Customer churn is one of the biggest challenges in the telecom industry. This project digs into **7,043 customer records** to uncover churn drivers across demographics, service types, contract plans, and usage patterns — all visualized in an interactive Excel dashboard.

---

## 🗂️ Dataset Summary

| Field | Details |
|---|---|
| **Total Customers** | 7,043 |
| **Churned Customers** | 1,869 (~26.5%) |
| **Features** | 37 columns |
| **Geography** | California, USA |

### Key Fields
- **Demographics**: Gender, Age, Marital Status, Number of Dependents
- **Services**: Phone, Internet Type (Fiber Optic / DSL / Cable), Streaming, Security
- **Financials**: Monthly Charge, Total Revenue, Total Refunds
- **Churn Info**: Customer Status, Churn Category, Churn Reason

---

## 🔍 Key Insights

### 📉 Churn Overview
| Status | Count | % |
|---|---|---|
| Stayed | 4,720 | 67% |
| **Churned** | **1,869** | **27%** |
| Joined | 454 | 6% |

### 🏆 Top Churn Reasons
1. 🥇 **Competitor** — 841 customers (competitors offered better devices/deals)
2. 😤 **Dissatisfaction** — 321 customers (product & network issues)
3. 😠 **Attitude** — 314 customers (poor customer service experience)
4. 💰 **Price** — 211 customers (too expensive)

### 📶 Internet Type Distribution
- **Fiber Optic**: 3,035 customers (most popular — also highest churn risk)
- **DSL**: 1,652 customers
- **Cable**: 830 customers
- **No Internet**: 1,526 customers

### 📄 Contract Type
- **Month-to-Month**: 3,610 → Highest churn risk
- **Two Year**: 1,883 → Most loyal customers
- **One Year**: 1,550

---

## 💡 Business Recommendations (from the data)

1. **Target Month-to-Month customers** with loyalty incentives to upgrade to annual contracts
2. **Investigate Fiber Optic churn** — despite being the most popular, it shows the highest dissatisfaction
3. **Address competitor threats** proactively — 45% of churners left for a competitor
4. **Improve customer service attitude** — 314 customers left due to poor interactions
